Welcome to TheOpenCD!

Este disco contiene una compilación de Software Libre de alta calidad para Windows. Fue concebido como una primera introducción al mundo del Software Libre. También fueron incluidos accesorios relevantes, documentación y explicaciones paso a paso. Siéntase libre de utilizar este software en varias computadoras y hacer copias para sus amigos. ¡Este software es libre!
