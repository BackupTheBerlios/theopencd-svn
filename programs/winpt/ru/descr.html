WinPT 1.0

Windows Privacy Tools (WinPT) это коллекция программ для дигитального шифрования, и подписи документов, электронной почты, и других файлов. Программа использует протокол GnuPG для шифрования, который совместим с OpenPGP совместимыми программами. С этой программой вы можете создать набор публичных и личных ключей, которые потом вы сможете использовать. Публичные ключи можно распространять свободно, для материала который предназначен только для ваших глаз. Потом для того чтобы открыть послание, используйте личный ключ (который вам лучше оберегать как зеницу ока) для дешифрования послания. В похожем стиле, вы можете подписаться используя ваш личный ключ, для подписей которые могут быть потом проверенны юзерами с подходящим публичным ключём. На диске есть детальные инструкции о WinPT в формате PDF.
