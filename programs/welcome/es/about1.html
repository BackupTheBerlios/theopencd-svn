El Software Libre

El Software Libre ( ver nota del traductor ) es proporcionado con una licencia que le da al usuario final el derecho a usarlo libremente para uso privado o comercial. Asimismo el derecho a inspeccionar y hasta modificar su código fuente. Usted puede entregar or vender la versión original que recibió, o una con modificaciones propias, siempre y cuando entregue el código fuente modificado para que otros puedan beneficiarse de los cambios que haya hecho. Esta última cláusula protege el trabajo del desarrollador de explotaciones injustas por terceros, a la vez que permite liberar el código fuente a la comunidad. No es necesario que usted pague por propiedad intelectual a los autores anteriores, y aún puede cobrar dinero por la venta de Software Libre. Este disco puede haber sido entregado a usted por un amigo; o vendido por una pequeña tienda, y ambos casos están permitidos. Tenga en mente que hay muchas licencias diferentes, cada una de las cuales tiene sus propias condiciones. Si planea modificar o distribuir uno de los programas en el disco, por favor verifique la licencia que se aplica al mismo antes de comenzar para estar seguro de sus derechos y obligaciones.

Si usted no tiene interés en el código fuente, puede que se pregunte porqué la disponibilidad del mismo es importante. Una respuesta es que permite a un observador externo verificar la verdadera funcionalidad del programa, con lo que usted puede confiar en que el programa trata sus datos privados con respeto. Un problema del software propietario es que utiliza formatos cerrados para guardar sus datos. Si el desarrollador del software dejar de soportar viejas versiones de un software propietario, usted puede verse forzado a una actualización a nuevas versiones con un costo significativo, para poder acceder a sus datos. Esto no ocurre con el Software Libre, porque con el código fuente para abrir y guardar archivos a su disposición, un tercero puede fácilmente crear un filtro de conversión para la próxima generación de software, asegurando que sus datos estarán siempre disponibles. Finalmente, la liberación de código fuente ha generado grandes comunidades de desarrolladores voluntarios quienes han proporcionado al mundo, software de alta utilidad y enteramente libre como Linux, Open Office y Mozilla. Estos están disponibles en forma gratuita para las escuelas o cualquier otro que no disponga de un gran presupuesto para software. Así, usted puede apreciar que el Software Libre es importante para todos, no solo para los desarrolladores.
