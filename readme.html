Welcome to TheOpenCD!

This disc contains a compilation of high quality Open Source software for Windows. It is intended as a first introduction to the rich world of Open Source. Also includes are relevant add-ons, documentation and tutorials. Feel free to use the programs on multiple computers and make copies for your friends. This software is free!
