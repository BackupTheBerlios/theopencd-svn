Welcome to TheOpenCD!

Deze schijf bevat een verzameling kwalitatief hoogwaardige Open Source programma's voor Windows. Het is bedoeld als een eerste kennismaking met de rijke wereld van Open Source Software. Tevens zijn relevante toevoegingen, documentatie en cursussen bijgevoegd. Gebruik het vooral op meerdere computers, en geef de CD door aan je vrienden. Deze software is vrij!

The programs on this disc are all distributed under an Open Source License (OSI approved), which allows you to freely use and distribute it. You may even change the programs using the source code, we make available and distribute your own modified versions, provided you then in turn make the source code available, and give appropriate credit to past contributors.

- TheOpenCD Team
